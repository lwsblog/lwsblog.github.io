利用Hexo+GitHub搭建免费的个人博客

这是一篇关于搭建个人博客的文章，图文详解搭建过程，本文使用Windows10 x64 2004(19041.207)系统演示。穷得只能在虚拟机里玩玩Macintosh

1. 众多疑问

1.1 为什么要搭建个人博客

​ ——个人博客可以随心所欲地写你想写的东西，不受第三方网站所限制，此外，自由度高，可以个性化成你想要的样子。

1.2 Github是什么？

​ ——GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。

1.3 Hexo是什么？

​ ——Hexo是当下主流的、简洁的、功能强大的博客框架。

​ 2.直接输入你的用户名、电子邮箱和密码，注意起用户名时最好慎重，因为你的博客的地址将是username.github.io

2.1.3 创建存储库

​ 1.进入GitHub官网，登录，点击右上方你头像右边的那个向下的三角形，选择Your repositories点击内个绿绿的New，会跳出来一个配置界面，这么选:

​ 注意：请务必勾选Initialize this repository with a README，并且仓库名就是我画两个红杠杠的那一行要用如下格式：username.github.io注：username是你的用户名。

​ 因为我事先注册过一遍，所以会出现错误提示。最后点击内个绿绿的Create repository。至此，你的存储库就创建完成了！
